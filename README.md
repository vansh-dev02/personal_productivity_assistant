# 🚀 Personal Productivity Assistant

A Python-based productivity management application that helps users manage tasks, notes, and track their productivity.

## ✨ Features

### User System
- Create account
- Login system
- User-specific data storage

### Task Management
- Add tasks
- View tasks
- Complete tasks
- Delete tasks
- Priority system
- Task date tracking

### Notes Management
- Create notes
- View notes
- Store personal notes

### Productivity Report
- Total tasks
- Completed tasks
- Pending tasks
- Productivity percentage


## 🛠️ Technologies Used

- Python
- Object Oriented Programming (OOP)
- JSON
- File Handling

## ▶️ How To Run

Clone the repository:

```
git clone https://github.com/vansh-dev02/personal_productivity_assistant.git
```

Go inside folder:

```
cd personal_productivity_assistant
```

Run:

```
python src/main.py
```
